# Problem statement
I need a model for predicting the team score of a wrestler based on his current attributes going into the ncaa tournament

# Dataset
2023-2025 ncaa tournament data + Matkings.com salary cap fantasy roster information

# Approach
Feature engineering from ranking, seeding, experience, and team context. Trained regression models to estimate expected tournament points.

# Models
Linear Regression
Ridge Regression
XGBoost / GradientBoostingRegressor

# Results
